# **ATS-Ready Resume Parser**

A lightweight, browser-based tool that simulates how an Applicant Tracking System (ATS) parses, tokenizes, and categorizes resume content. This project helps you evaluate keyword coverage, skill grouping, and text structure in a way that mirrors automated screening systems.

## **Live Demo**

**[https://oyoai.github.io/resume-ats/](https://oyoai.github.io/resume-ats/)**

## **Project Structure**

```
index.html      – Main interface  
script.js       – Parsing and analysis logic  
style.css       – Layout and visual styling  
```

## **Run Locally**

```bash
git clone https://github.com/oyoai/resume-ats.git
cd resume-ats
open index.html    # or double-click index.html
```

## **Project Status**

Actively expanding profiles, keyword groups, and interface functionality to improve ATS realism and coverage.
